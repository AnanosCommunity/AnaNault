Remote Signing

An extremely secure way of transacting on the Nano network without exposing your private keys to the Internet, ensuring only non-sensitive data is transferred between devices.
A full security and user guide is found here.

STEP 1 - Create Block (Online Device)

Create a SEND, RECEIVE or CHANGE block from a "watch-only" account. No login is needed.

STEP 2 - Sign Block (Offline Device)

Scan the unsigned QR code from Step 1 OR paste the block below. Sign it using a key owning the account from step 1.
Nault can optionally be set in Offline Mode from the Server Configuration to avoid network notifications.

STEP 3 - Process Block (Online Device)

Scan the signed QR code from Step 2 OR paste the block below. Confirm and send it to the live network!
