Remote Signing

Remote signing allows you to transact on the Nano network with extra security.

  1. Create a block online using the latest information from the network.
  2. Sign the block hash on an offline device using a private key corresponding to the account from step 1.
  3. Embed the signature and process the final block online.

This way the private key avoid any exposure to the Internet and only non sensitive data is tranferred between the devices.

STEP 1 (Online Device)

Start with an account in "watch-only" mode. No login is needed.
Remote Signing will be enabled for you to create new blocks.

GO

STEP 2 (Offline Device)

Scan the unsigned QR code from Step 1.
Confirm and Sign it using any of the methods provided. Seed, Mnemonic, Private key or Ledger hardware wallet. Proof of Work is optional.

STEP 3 (Online Device)

Scan the signed QR code from Step 2.
Confirm and Send the block to the live network! Proof of Work will be made if not done in Step 2.
