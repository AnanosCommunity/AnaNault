Remote Signing

An extremely secure way of transacting on the Nano network without exposing your private keys to the Internet, ensuring only non-sensitive data is transferred between devices.

  1. Create a block online using the latest information from the network.
  2. Sign the block hash on an offline device using a private key corresponding to the account from step 1.
  3. Return the signature and process the final block online.

STEP 1 (Online Device)

Start with an account in "watch-only" mode. No login is needed.

STEP 2 (Offline Device)

Scan the unsigned QR code from Step 1 OR paste the block text below.
